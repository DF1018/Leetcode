# Easy
### Given two strings s and t, return true if t is an anagram of s, and false otherwise.


```
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        if not len(t) == len(s):
            return False

        sorted_s,sorted_t = sorted(s) , sorted(t)
        if sorted_s == sorted_t:
            return True
        else:
            return False
```
