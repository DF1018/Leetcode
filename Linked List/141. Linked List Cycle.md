```
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        slow = head
        fast = head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```

### 假設在環中，快指針 Fast 與慢指針 Slow 之間的距離為 n。
### 每當兩者各前進一次，S 向前移動 1 個節點， (n + 1)
### 而 F 向前移動 2 個節點。 (n + 1 - 2)
### 由於 F 的移動速度比 S 快 1， 
### 因此兩者之間的距離會以每次 1 個節點的速度縮短。(n - 1)
### 當距離減少至 0 時，兩者便會相遇。
