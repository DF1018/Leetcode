neetcode
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        maxP = 0
        l, r = 0, 1
    
        while r < len(prices):
            if prices[l]<prices[r]:
                profit = prices[r] - prices[l]
                maxP = max(maxP, profit)
            else:
                l=r
            r+=1
        return maxP
```

leetcode大神
```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        lowest = float('inf')
        highest = 0
        max_profit = 0
        for price in prices:
            if price < lowest:
                lowest = price
                highest = price
            if price > highest:
                highest = price
                if highest - lowest > max_profit:
                    max_profit = highest - lowest
        
        return max(0, max_profit)
```
