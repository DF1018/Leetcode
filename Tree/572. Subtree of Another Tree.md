# Subtree of Another Tree — 思路總整理（美化版）

## subRoot 是 root 的 subtree 可能的情況

1. **subRoot 與 root 的整棵樹完全相等**
2. **subRoot 是 root.left 的 subtree**
3. **subRoot 是 root.right 的 subtree**

---

## 判斷兩棵樹（root 與 subRoot）是否「相等」的條件

### ✔️ 1. 若 root 為空，則 subRoot 必須也為空

### ✔️ 2. root 與 subRoot 都不為空，且兩棵樹的結構與值都完全相同

### ✔️ 3. root 不為空時，也可能 subRoot 是 root.left 的 subtree

### ✔️ 4. root 不為空時，也可能 subRoot 是 root.right 的 subtree

---

# neetcode

```python
class Solution:
    def isSubtree(self, s: Optional[TreeNode], t: Optional[TreeNode]) -> bool:
        if not t :
            return True 
        if not s :
            return False
        
        if self.sameTree(s, t):
            return True
        
        return (self.isSubtree(s.left, t) or
                self.isSubtree(s.right, t))
               
    def sameTree(self, s, t):
        if not s and not t:
            return True
        if s and t and s.val == t.val:
            return (self.sameTree(s.left, t.left) and
                    self.sameTree(s.right, t.right))
        return False
```

```python
class Solution:
    def isSubtree(self, root: Optional[TreeNode], subRoot: Optional[TreeNode]) -> bool:
        if not subRoot:
            return True
        
        elif not root and subRoot:
            return False

        else:
            return (self.isIdentical(root, subRoot) or
                    self.isSubtree(root.left, subRoot) or
                    self.isSubtree(root.right, subRoot))
            
    def isIdentical(self, s, t):
        if not s and not t:
            return True
            
        if not s or not t or s.val != t.val:
            return False

        return (self.isIdentical(s.left, t.left) and
                self.isIdentical(s.right, t.right))
```
